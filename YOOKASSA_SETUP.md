# 💳 Подключение ЮКассы — пошаговая инструкция

## Шаг 1 — Регистрация на ЮКассе

1. Зайди на **yookassa.ru** → нажми «Подключиться»
2. Заполни данные ИП (ИНН, расчётный счёт, данные паспорта)
3. Подпиши договор электронно (через Госуслуги или SMS)
4. Ждёшь проверку — обычно 1–3 рабочих дня

## Шаг 2 — Получение ключей

1. После одобрения: **yookassa.ru** → Личный кабинет
2. Настройки → **Ключи API**
3. Скопируй:
   - **shopId** (число, например `123456`)
   - **Секретный ключ** (вида `test_...` или `live_...`)

> ⚠️ Сначала работай с тестовым ключом (`test_...`), потом переключи на боевой

## Шаг 3 — Добавить ключи в Vercel

1. Зайди на **vercel.com** → твой проект `jk-brand`
2. Settings → **Environment Variables**
3. Добавь две переменные:

| Name | Value |
|------|-------|
| `YOOKASSA_SHOP_ID` | твой shopId |
| `YOOKASSA_SECRET_KEY` | твой секретный ключ |

4. Нажми **Save**
5. Зайди в Deployments → нажми **Redeploy** (чтобы переменные применились)

## Шаг 4 — Настройка в ЮКассе

1. В кабинете ЮКассы: **Настройки → HTTP-уведомления**
2. Добавь URL: `https://твой-домен.ru/api/payment-webhook` (опционально — для авто-подтверждений)

## Шаг 5 — Тест оплаты

Используй тестовую карту:
- Номер: `5555 5555 5555 4444`
- Срок: любой в будущем
- CVV: любые 3 цифры
- 3DS-код: `12345678`

✅ Если деньги "списались" в тестовом режиме — всё работает!
Меняй ключ на боевой и запускай.

## Комиссия ЮКассы
- Карты: **3.5%** с каждого платежа
- СБП: **1%** (выгоднее!)
- Выплата на расчётный счёт: раз в рабочий день
